### 功能:

- [x] emoji 使用了最基本的集合，支持任何 App；（ 均不会出现乱码情况）

- [x] 快捷键插入后，光标仍处于输入框中；

- [x] 颜值尚能接受；
  _全部的 emoji 使用了 Google Android 7.0_

- [x] 可作为独立窗口存在；

- [x] 支持关键字（英文）查询匹配 emoji；

- [x] 支持快捷键呼出；

### 已知问题：

- iFrame 无法使用  `dropdown` 模式;

- 非标准 `<input> / <textarea>` 类标签无法使用快捷键 `::  ` 呼出；

- 当为 `window` 独立使用状态时，插入功能失效；
